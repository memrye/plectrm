# Plectrm
![image of plectrm logo](https://raw.githubusercontent.com/memrye/plectrm/refs/heads/main/misc/plectrm256px.png)

---
Plectrm is a (guitar) tablature editor, designed like a text editor. It is open source and written with Javascript. It is distributed under the GNU General Public License v3.
https://plectrm.github.io

## Contributions

If you'd like to contribute to Plectrm, please feel free to submit a PR. You can also message me wherever you can find me if you have any questions <3
